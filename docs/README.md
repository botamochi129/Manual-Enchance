# Manual Enchance

**Manual Enchance** は、[Minecraft Trainsit Railway](https://modrinth.com/mod/minecraft-transit-railway) の手動運転機能を強化し、Nemo's Transit Expansion (NTE) のスクリプト機能で使用できるメソッドを追加するアドオンmodです。

> [!TIP]
> このMODは現在開発中のため、仕様が変更される可能性があります。

## 対応バージョン
このmodは、Minecraft Transit Railway 3.2.2 をサポートしています。

**MTR4には対応していません。**

## 対応言語
現時点では以下の言語に対応しています。

- 日本語(ja_jp)
- 英語(en_us)

## 主な機能
- 手動運転時のノッチ段階を従来の5段階(P2～B2)から、15段階(P5～B8+EB)に拡張します。
- MTR標準の手動運転HUDを、独自のHUDに変更します。
- パンタグラフの操作機能を実装します。(開発中)
- 運転中に警笛を鳴らすことが出来ます。
- リバーサー(F/N/B)を追加します。
- 手動で操作できる方向幕を追加します。
- KATO/ZUIKIマスコンによる運転が可能になります。
- NTEのスクリプト機能で利用できるメソッドを追加します。
