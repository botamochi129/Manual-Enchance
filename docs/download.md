# ダウンロード

## 配布場所
以下からダウンロードできます。

* **[Modrinth](https://modrinth.com/mod/manual-enchance)**
* **[GitHub Releases](https://github.com/botamochi129/Manual-Enchance)**

## 依存関係
動作には以下のMODが必須です。バージョンが合っているか確認してください。

| MOD名 | 推奨バージョン |
| :--- | :--- |
| Minecraft Transit Railway | 3.2.2 |
| Architectury API | 各マイクラバージョン用最新版 |
| NTE / ANTE | 最新版 |

> [!WARNING]
> MTR 4.xには対応していません。
